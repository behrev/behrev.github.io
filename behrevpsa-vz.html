behrev/PSA-VZ 

behrev Januar 2, 2025

Mit der Anwendung hier kann von einem Anfangsdatum, welches hier auf den 17.09.2017 mit dem PSA Wert von 0,02 voreingestellt ist, die Verdopplungszeit in Monaten berechnet werden.

Jeder Wert wird mit Datum, PSA-Wert und Verdopplungszeit in Monaten zwischen gespeichert. Mit dem Button „Sichern“ kann die Tabelle als .json File gespeichert werden, da die Wert alle nach dem schließen gelöscht werden. Nach dem erneuten Aufruf kan das .json File wieder geladen werden. Das file kann auch mit einem Editor manuel bearbeitet/erweitert werden.

Die Diagramme werden mit einem externen Addon gezeichnet. Die Quelle ist hier.
https://cdn.jsdelivr.net/npm/chart.js
